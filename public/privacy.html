Politica de privacidad

Ultima actualizacion: 21 de mayo de 2026.

Volver Terminos Cookies

1. Principio de minimizacion

La pagina esta disenada para pedir y conservar la menor cantidad de datos necesaria. El archivo se usa para extraer o escribir metadata tecnica y no se almacena como contenido permanente.

2. Datos que se procesan

  • Nombre del archivo, tamano, extension, tipo MIME declarado o detectado y fecha de analisis.
  • Metadata tecnica detectada por el analizador, segun el formato del archivo.
  • Campos de metadata que escribes en el editor, solo para generar la copia solicitada.
  • Si usas una cuenta: usuario, correo, sesiones, historial de analisis y actividad basica de seguridad.
  • Telemetria tecnica administrable: IP observada por el servidor/proxy, navegador/dispositivo inferido por user-agent, ruta solicitada, fecha, idioma del navegador, pais reportado por Cloudflare cuando exista y eventos de seguridad.

3. Datos que no busca conservar

  • No guarda archivos completos como biblioteca, hosting o respaldo.
  • No crea enlaces publicos ni descargas compartibles.
  • No usa la pagina para analitica invasiva, anuncios ni perfilado oculto.
  • No solicita permisos del navegador como camara, microfono o ubicacion.
  • No intenta descubrir ubicacion exacta, identidad real, contactos, archivos locales adicionales ni datos fuera de lo que el navegador y el proxy entregan para operar la solicitud.

4. Seguridad de sesion

Las sesiones de cuenta usan cookies HttpOnly con SameSite=Lax. La contrasena se almacena como hash con sal e iteraciones. El historial de cuenta se limita a informacion tecnica del analisis, no al contenido completo del archivo.

5. Retencion y control

Los archivos temporales se eliminan al terminar el procesamiento o ante error. Los registros asociados a una cuenta pueden mantenerse para mostrar historial y sesiones. La telemetria tecnica puede revisarse desde el panel administrativo para seguridad, abuso, soporte y bloqueo de IPs o usuarios. Puedes cerrar sesiones desde la pagina de cuenta.

6. Terceros

La interfaz evita scripts externos innecesarios. Las herramientas tecnicas usadas para procesar metadata se ejecutan para cumplir la accion solicitada y no convierten el archivo en contenido publico.
